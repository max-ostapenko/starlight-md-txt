## Features

<CardGrid idx="features">
  <Card title="Agent Friendly" icon="open-book">
    Exposes each documentation page as raw, clean Markdown at `/page.md.txt`.
  </Card>
  <Card title="AST-based Transpilation" icon="setting">
    Uses unified/remark parsers to translate complex Starlight/JSX components into native Markdown.
  </Card>
  <Card title="Selective Frontmatter" icon="document">
    Only retains `title` and `description` to keep LLM context clean and efficient.
  </Card>
  <Card title="Draft Support" icon="pencil">
    Mirrors Starlight's draft behavior or can optionally include draft pages in builds.
  </Card>
</CardGrid>
